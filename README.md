# 📊 Mobile Sales Dashboard — Power BI

## 🔍 Project Overview
An interactive 3-page Power BI dashboard analyzing 
mobile sales performance across India from 2021 to 2024, 
covering 5 major brands — Apple, Samsung, OnePlus, Vivo & Xiaomi.

## 📌 Key Features
- 3-page interactive dashboard with navigation buttons
- KPI Cards — Total Sales (₹769M), Quantity, Transactions, Avg Price
- Geographic map — city-wise sales across India
- DAX Measures — MTD, QTD, YTD, Same Period Last Year
- Year-over-Year comparison charts
- Dynamic slicers — Brand, Mobile Model, Payment Method

## 💡 Key Findings
- Delhi recorded the highest sales among all cities
- 2023 was the strongest performing year overall
- Apple led revenue at ₹161M due to premium pricing
- UPI was the most preferred payment method (26.36%)
- July had highest units sold (1,700) — mid-year demand surge
- Monday & Friday are peak sales days

## 🛠️ Tools Used
- Microsoft Power BI Desktop
- DAX (Data Analysis Expressions)
- Microsoft Excel (Data Cleaning)
